# netgear-ex8000-restart
Restart your Nighthawk X6S Tri-Band Range Extender from command line

## Install
```
npm i
```

## Run
```
EMAIL=YOUR_EMAIL PASSWORD=YOUR_PASSWORD node index.js
```
